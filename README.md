# Enslin's Beauty Parlor - Professional Beauty System

## Client Mock Project

This is a professional mockup developed for demonstration purposes to showcase capabilities to potential clients. All business names, data, and branding are fictional and used for illustrative purposes only.

## Project Overview

A comprehensive, full-stack beauty salon and academy management system designed to showcase modern web development capabilities. This mock project demonstrates a complete business solution with booking systems, service management, and academy course enrollment.

## Technologies Used

### Frontend
- React 18 (CDN delivery)
- Modern CSS3 with Grid & Flexbox
- CSS Custom Properties (Variables)
- CSS Animations and Transitions
- Responsive Web Design
- Mobile-First Approach

### Backend
- Django 4.2 (Python)
- Django REST Framework
- SQLite Database
- Python Decouple for environment variables
- WhiteNoise for static files
- CORS Headers for API security

### Development & Deployment
- Git for version control
- Virtual Environment (venv)
- Gunicorn (production WSGI server)

## Demo Features Showcased

- Beauty Services Booking System
- Professional Academy Course Management
- Service & Course Combo Packages
- Mobile-First Responsive Design
- iOS-Inspired Animations
- Real-time Booking Management
- Professional Image Gallery
